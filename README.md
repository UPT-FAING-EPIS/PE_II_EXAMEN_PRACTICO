## Mejoras realizadas

### 1. Manejo de errores y validaciones en el login y formularios
Se implementó un sistema de mensajes claros para errores y validaciones en el inicio de sesión y formularios principales, mejorando la experiencia del usuario y la seguridad.

### 2. Paginación en la lista de usuarios
Se añadió paginación en la gestión de usuarios, permitiendo navegar entre páginas de 10 usuarios, optimizando el rendimiento y la usabilidad cuando existen muchos registros.

---

## Descripción

Este repositorio contiene el avance del sistema PETI con las mejoras solicitadas para el examen práctico de la Unidad II. Se mejoró la experiencia de usuario mediante validaciones y mensajes de error claros, y se optimizó la gestión de usuarios con paginación.
